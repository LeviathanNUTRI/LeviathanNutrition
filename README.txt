LEVIATHAN NUTRITION — GUÍA DE IMÁGENES
========================================

Todas las imágenes del sitio están preparadas con rutas fijas. Para agregar
las tuyas, simplemente coloca el archivo con el MISMO nombre en la misma
carpeta (reemplazando el que exista o creando la carpeta si no existe).
No necesitas tocar el código HTML/CSS/JS.

Si un archivo no existe o no carga, la página muestra automáticamente un
ícono de reemplazo (placeholder) en su lugar, así que nunca se rompe el
diseño mientras vas agregando tus fotos reales.

------------------------------------------------------------
NOTAS
------------------------------------------------------------
- Puedes usar .jpg, .png o .webp; solo mantén el mismo nombre de archivo
  o actualiza el atributo "src" correspondiente en index.html.
- Los precios, nombres y categorías de los productos son de ejemplo:
  edítalos directamente en index.html (busca la clase "product-card").
- El carrito, los filtros, el formulario de contacto y el newsletter
  funcionan con JavaScript en el navegador (sin backend). Para conectarlos
  a un sistema de pagos o correo real, se necesita integrar un backend.
